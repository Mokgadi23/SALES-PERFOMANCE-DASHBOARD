# Sales & Profit Performance Dashboard

## 📊 Project Overview

This project analyzes business sales data to uncover key insights into revenue trends, product performance, regional performance, and profitability. The dashboard was built using Power BI.

## 🛠 Tools Used

* Power BI
* Python (for data cleaning)
* Excel/CSV dataset

## 📈 Key Insights

* Sales show a steady increase over time
* Technology is the top-performing category
* Some products generate high sales but low profit
* West and East regions perform better than others
* Consumer segment contributes the most sales

## 💡 Recommendations

* Focus on high-performing categories like Technology
* Review discount strategies to improve profit margins
* Invest more in high-performing regions
* Target the Consumer segment for growth

## 📁 Files Included

* Power BI Dashboard (.pbix)
* Dataset (optional)
* Dashboard screenshots

## 📌 Conclusion

The analysis highlights opportunities to improve profitability while maintaining strong sales growth.
